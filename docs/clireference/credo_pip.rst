.. _credo_pip:

credo pip
---------

Retrieves a python package and its dependencies.

Synopsis
~~~~~~~~


Retrieves a python package and its dependencies.

::

  credo pip [flags]

Examples
~~~~~~~~

::


  Install a pip package:
  	credo pip numpy

  Install a pip package pinning it to a version:
  	credo pip numpy==1.26.0


Options
~~~~~~~

::

  -h, --help   help for pip
